Prototype

Prototype => Whenever we create a object or string or function js engine attaches the object to it so we can access the property of that object.

Prototype Chain =>

1) Arr.prototype give the same results arr.__proto__,

2) Arr.__proto__.__proto__ gives the same results as Object.prototype (its results is also a object so we can get its prototype also (see in next line)),

3) Arr.__proto__.__proto__.__proto__ gives the results null when we reach to tha null so that is the end of the chain, it is known as prototype chain or prototypal inheritance.
